# Welcome

How to use this plugin?
you can use in several ways:


# PHP 

<code>
$options = get_option( 'meals-donated', array() );
$meals = $options['number_of_meals'];
return	 $meals;
</code>

# WordPress Shortcode

this is the shortcode <code>[meals]</code>

# HTML variable

use <code>{$meals}</code> everywere you need it.
